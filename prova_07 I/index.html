Cursos Front End

HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página. Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos

PHP

PHP (Hypertext Preprocessor) é uma linguagem de programação de script open source, muito utilizada no desenvolvimento de páginas web. O PHP é embutido no código HTML, o que permite que os desenvolvedores web criem páginas dinâmicas de forma rápida.
